▶️ VastVideo-Go
极速多源影视聚合播放器 · 开源 · 全端适配 · 极速体验
项目简介
VastVideo-Go 是一款基于 Go 后端和现代前端技术开发的多源影视聚合播放器，支持 PC 与移动端自适应，聚合主流资源站，极速搜索、高清播放，界面美观，体验流畅。适合技术爱好者、家庭用户、影视发烧友等多种场景。
主要功能亮点
多源聚合
支持主流资源站一键聚合，自动切换、优选资源，内容丰富。
极速搜索
全局异步搜索，支持多源并发，结果秒级返回，体验极致流畅。
高清播放
支持4K/1080P/720P等多分辨率，自动识别，原画播放不卡顿。
全端适配
移动端、PC端自适应布局，触控/鼠标操作均友好，响应式设计。
开源可定制
完全开源，支持二次开发、私有部署，社区活跃。
安装与使用
1. 获取源码： git clone https://github.com/VastPools/VastVideo-Go.git 2. 启动后端（Linux/macOS）： cd VastVideo-Go && ./build.sh && ./vastvideo-go 3. Windows 直接运行： 双击 vastvideo-go.exe 或运行 start.bat 4. macOS 直接运行： 双击 vastvideo-go 或在终端运行 ./vastvideo-go 5. Docker 一键运行：
x86 架构：
docker run -d --name vastvideo-go -p 8228:8228 vastpools/vastvideo-go:2.0.0
ARM 架构（如树莓派、部分 Mac）：
docker run -d --name vastvideo-go -p 8228:8228 vastpools/vastvideo-go:2.0.0 6. 访问前端： 浏览器访问 http://:8228 7. 移动端体验： 手机浏览同一地址，自动适配移动端界面
* 详细配置与高级玩法请参考项目 README 或社区文档。
